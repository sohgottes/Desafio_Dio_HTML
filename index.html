O que é HTML?

HTML Linguagem de Marcação de Hipertexto) é a base da construção de qualquer site. Ele define a estrutura de uma página da web utilizando elementos chamados "tags". Essas tags são usadas para criar títulos, parágrafos, listas, links, imagens e outros elementos visuais e estruturais.

HTML não é uma linguagem de programação, pois não possui lógica ou variáveis. Seu principal propósito é organizar e estruturar o conteúdo de forma semântica, permitindo que navegadores e leitores de tela interpretem corretamente a informação.

Combinado com CSS (folhas de estilo) e JavaScript (interatividade), o HTML se torna ainda mais poderoso para o desenvolvimento web.

Tags Semânticas

Tags semânticas são aquelas que descrevem claramente o significado do conteúdo nelas contido, tanto para o navegador quanto para o desenvolvedor. Exemplos incluem <main>, <section>, <article> e <aside>.

  • <main>: Delimita o conteúdo principal da página.
  • <section>: Agrupa conteúdos relacionados, como capítulos ou seções.
  • <article>: Representa uma composição independente, como postagens ou artigos.
  • <aside>: Usado para conteúdo complementar, como barras laterais.

Títulos Hierárquicos

  1. Título principal da página

  2. Subtítulo ou seção

  3. Subseção de h2

  4. Detalhamento de h3

  5. Níveis adicionais
  6. Notas ou informações de menor importância

Formatações de Texto

  • <strong>: Dá ênfase em negrito
  • <i>: Itálico
  • <u>: Sublinhado
  • <mark>: Texto destacado
  • <small>: Texto menor
  • <del>: Texto riscado
  • <sub>: Subscrito, ex: H2O
  • <sup>: Sobrescrito, ex: x2
  • ONU: Usando <abbr>
  • Texto com a tag obsoleta <font>

Dicas Finais

  • Organize seu código com indentação consistente.
  • Use tags semânticas para dar clareza ao seu HTML.
  • Comente o código para facilitar a manutenção e entendimento.
  • Evite usar tags obsoletas como <font> em projetos reais.
  • Pratique muito e experimente criar páginas reais para fixar o conteúdo.
